识别资源

设计uri并选择合适的http方法和返回码

  • 合理的利用分隔符表述资源之间的层次关系
    • 例如 ： /user/{id}/
  • 使用逗号或者分号表述非层次关系（实际应用很少用）
  • 使用连接字符“-” “_” 改善长路径名称可读性
  • 避免出现一些后缀名
  • 使用& 区分参数
